# **Tic-Tac-Toe**

A stylish, web-based implementation of the classic Tic-Tac-Toe game. This project focuses on a clean user experience, minimalist retro design, and real-time state tracking.

## **🚀 Features**

* **Customizable Players:** Edit player names (e.g., "Max" and "Manuel") to personalize the match.  
* **Dynamic State Logging:** Real-time feedback showing the exact coordinates of the last move.  
* **Responsive Board:** A $3 \\times 3$ grid designed with a vintage color palette and intuitive interactions.  
* **Turn Indicators:** Visual highlighting to show which player is currently active.

## **🛠️ Technologies Used**

* **Frontend:** React (suggested) / HTML5 & CSS3  
* **Styling:** Custom CSS with a focus on CSS Grid and Flexbox for the game board layout.  
* **Icons/Assets:** Custom stylized stone-texture icons for X and O.
